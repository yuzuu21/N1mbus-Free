## Overview

**Nimbus is a high-performance utility client designed for Minecraft 1.21.8.**

It specializes in silent rotations, smooth rendering, and advanced anti-cheat bypasses, providing powerful features in a lightweight package. With our proprietary **Silent Rotation** and **GCD Smoothing** technology, Nimbus sends natural packets simulating human input to the server while maintaining a premium, responsive feel for the player.

> [!NOTE]
> This client is exclusively for the Fabric environment. Forge and NeoForge are not supported.

## Features

### Combat

| Module | Description |
| :--- | :--- |
| **KillAura** | Features GCD-based smoothing, silent rotations, and adaptive aim points for advanced targeting. |
| **Velocity** | Knockback reduction that stays in sync with the server to minimize rubberbanding. |

### Visuals

| Module | Description |
| :--- | :--- |
| **3D World ESP** | Smooth box ESP that perfectly tracks entity movement using frame interpolation. |
| **2D Smooth ESP** | Screen-space smoothed boxes with real-time health indicators. |
| **FullBright** | Keeps the world bright at all times without the need for Night Vision potions. |

### Movement & Misc

| Module | Description |
| :--- | :--- |
| **Sprint / Step** | Essential utilities to enhance movement comfort and agility. |
| **NoFall** | Prevents fall damage through packet-level state management. |
| **Modern ClickGUI** | A sleek, intuitive interface for real-time configuration. |

## Installation

> [!IMPORTANT]
> **Java 21** and **Fabric Loader** are required before installation.

**1.** Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.8.

**2.** Place `nimbus-1.0.0-Pre.jar` in your `.minecraft/mods` folder.

**3.** Launch the game and enjoy.

> [!WARNING]
> Using this client may violate server rules. Always check the terms of service of the server you are playing on.
